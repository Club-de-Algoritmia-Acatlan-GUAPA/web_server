          Sum of Two Values

          Tiempo Limite: 1 s

          Limite de memoria: 2 MB

          Problem

          The first input line has two integers nnn and xxx: the array size and the target sum. The second line has nnn integers a1,a2,...,ana_1,a_2, ...,a_na1​,a2​,...,an​: the array values.
          1≤x≤1051 \leq x \leq 10^51≤x≤105
          • 1≤n≤2⋅1051 \leq n \leq 2 \cdot 10^51≤n≤2⋅105
          • 1≤x,ai≤1091 \leq x,a_i \leq 10^91≤x,ai​≤109

          Input

          You are given an array of nnn integers, and your task is to find two values (at distinct positions) whose sum is xxx.

          Output

          Print two integers: the positions of the values. If there are several solutions, you may print any of them. If there are no solutions, print IMPOSSIBLE.
